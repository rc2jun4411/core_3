1.2.7 Styling things based on state
= Notes =
  This has different states depending on whether it is 
unvisited, visited, being hovered over, focused via the keyboard,
  or in the process of being clicked (activated). You can use 
CSS to target these different states 

— the CSS below styles unvisited links pink and 
                       visited links green.

CSS:
a:link {
  color: pink;
}

a:visited {
  color: green;
}

a:hover {
  text-decoration: none;
}
